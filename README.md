# Kerala Tourism Website

## Overview

The Kerala Tourism Website is a responsive web project developed using HTML and Bootstrap. The website showcases the beauty, culture, and tourist attractions of Kerala, popularly known as **"God's Own Country."** It provides visitors with information about Kerala, its districts, famous destinations, and contact details through a clean and user-friendly interface.

## Features

* Responsive navigation bar
* Home page with image carousel
* Introduction to Kerala and its tourism attractions
* Detailed About Kerala page
* Frequently Asked Questions (FAQ) section
* District information with descriptions and location maps
* Contact page with contact details and inquiry form
* Mobile-friendly design using Bootstrap

## Technologies Used

* HTML5

* Responsive Web Design Principles

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Launch `index.html` in any modern web browser.
4. Navigate through the pages using the navigation menu.
